# Inflamed – HRV & Stress Prediction

This project explores how **Heart Rate Variability (HRV)** signals can be used to analyze and predict daily stress levels using machine learning techniques.

The goal of the project is to demonstrate how physiological data can be transformed into meaningful insights and predictive models.

---

# Project Overview

Stress detection using physiological signals is an important area in digital health and wearable technology.

In this project, HRV metrics are analyzed to understand patterns in stress levels and build models that can predict stress based on historical physiological data.

The project includes exploratory analysis and multiple regression models to evaluate predictive performance.

---

# Notebooks

## HRV.ipynb

This notebook focuses on exploring HRV data and understanding patterns related to stress.

Key steps include:

- Loading HRV dataset
- Data preprocessing and cleaning
- Exploratory data analysis (EDA)
- Visualizing HRV patterns
- Identifying relationships between HRV metrics and stress levels

---

## Inflamed-Stress.ipynb

This notebook focuses on building machine learning models to predict stress levels.

Models explored include:

- Linear Regression
- Ridge Regression
- Lasso Regression
- XGBoost

Techniques used:

- Feature selection
- Model training
- Model evaluation
- Comparing model performance
